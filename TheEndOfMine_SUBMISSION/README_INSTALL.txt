วิธีติดตั้ง TheEndOfMine

เวอร์ชัน APK: 1.0.2

1. ให้โหลดไฟล์ TheEndOfMine_PROJECT.apk ลงมือถือ Android
2. เปิดไฟล์ APK แล้วกด Install
3. ถ้ามือถือถามเรื่องความปลอดภัย ให้เปิด Allow from this source หรือ อนุญาตให้ติดตั้งแอปจากแหล่งนี้
4. ถ้าเคยติดตั้ง TheEndOfMine มาก่อน ให้ลบแอปเก่าออกก่อน แล้วค่อยติดตั้งไฟล์ APK ตัวนี้
5. เมื่อติดตั้งเสร็จ ให้เปิดแอปชื่อ TheEndOfMine
6. หน้าแรกจะเป็นภาพปกเกม ให้กด TAP TO START เพื่อเข้าเกม
7. ถ้าเครื่องเคยมีเซฟเก่าที่ทำให้เข้าไม่ได้ ให้ถอนการติดตั้งแอปเก่าหรือ Clear storage ก่อนติดตั้งใหม่

หมายเหตุ
- ไฟล์ที่ใช้ติดตั้งคือ TheEndOfMine_PROJECT.apk
- ไฟล์ TheEndOfMine_PROJECT.zip เป็น source code สำหรับส่งงาน ไม่ใช่ไฟล์ติดตั้งบนมือถือ
- APK นี้ทำมาสำหรับมือถือ Android จริงที่เป็น arm64-v8a ซึ่งมือถือ Android รุ่นใหม่ส่วนใหญ่ใช้ได้
- APK นี้ถูก build ใหม่ให้ฝัง runtime/assembly ครบสำหรับส่งให้เครื่องอื่นติดตั้ง ไม่ใช่ fast deployment APK
- ถ้าเปิดแล้วเด้ง ให้ถ่ายรูปหน้าจอหรือส่ง logcat ให้คนทำโปรเจคตรวจต่อ
